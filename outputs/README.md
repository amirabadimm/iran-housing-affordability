# Outputs

Notebook `01_housing_affordability_analysis.ipynb` writes reproducible analysis tables to `outputs/tables/`.

The CSV files in `outputs/tables/` are generated artifacts and are committed as lightweight result tables. Recreate or refresh them locally by running:

1. `notebooks/00_prepare_master.ipynb`
2. `notebooks/01_housing_affordability_analysis.ipynb`

Do not commit large intermediate files here.
