# Outputs

Notebook `01_housing_affordability_analysis.ipynb` writes reproducible analysis tables to `outputs/tables/`.

The CSV files in `outputs/tables/` are generated artifacts and are ignored by git. Recreate them locally by running:

1. `notebooks/00_prepare_master.ipynb`
2. `notebooks/01_housing_affordability_analysis.ipynb`

Only lightweight placeholders and documentation should be committed from this folder.
